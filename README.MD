## Installation

```sh
npm install --save-dev @vodyani/eslint-config
```

## Usage

**Add the `eslintConfig` configuration to `package.json`, then the inheritance relationship like this:**

```json
{
  "eslintConfig": {
    "extends": "@vodyani"
  }
}
```

## Dependencies

- eslint `v8`
- typescript-eslint `v5`
- eslint-plugin-import `v2`
